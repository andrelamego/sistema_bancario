# Overview do Projeto

O **Sistema Bancario** e um projeto academico de Java Web voltado para praticar modelagem de dados e regras de negocio em um cenario bancario.

## Objetivo

Disponibilizar uma aplicacao web para gestao de clientes, agencias e contas bancarias, com foco em:

- Arquitetura MVC em camadas (Controller, Service, Persistence e Model)
- Integracao com SQL Server via JDBC e stored procedures
- Regras de negocio implementadas no backend e no banco de dados

## Stack Tecnica

- Java 17
- Jakarta Servlet e JSP
- JDBC (jTDS)
- SQL Server
- Maven

## Escopo Funcional

O sistema cobre cadastro e autenticacao de clientes, administracao de agencias, abertura e manutencao de contas correntes/poupanca e suporte a conta conjunta.

## Contexto Academico

Projeto desenvolvido para fins didaticos na disciplina de Laboratorio de Banco de Dados (FATEC).
